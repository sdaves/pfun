    pfun.functions.curry(f)

    Get a version of f that can be partially applied

    Examples:

    >>> f = lambda a, b: a + b
>>> f_curried = curry(f)
>>> f_curried(1)
functools.partial(<function <lambda> at 0x1051f0950>, a=1)
>>> f_curried(1)(1)
2

    Parameters:

    Name Type Description Default
    f Callable

    The function to curry

    		 required

    Returns:

    Type Description
    Callable

    Curried version of f
