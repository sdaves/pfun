    pfun.maybe.maybe(f)

    Wrap a function that may raise an exception with a Maybe. Can also be used as a decorator. Useful for turning any function into a monadic function

    Examples:

    >>> to_int = maybe(int)
>>> to_int("1")
Just(1)
>>> to_int("Whoops")
Nothing()
